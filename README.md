# Student Dashboard Portal
Tulsa Job Corps — Student Academic & Trades Dashboard

A Google Apps Script web application that tracks student 
academic and trades program progress. Surfaces data through 
a dashboard with student profile cards, risk scoring, 
TABE tracking, and productivity tools.

## Structure
- dashboard/gs/ — Server-side Apps Script files
- dashboard/html/ — Client-side HTML/JS templates
- academic-tracker/ — Academic Tracker scripts (coming)
- docs/ — Design specifications and planning documents

## Data Sources
- SS_HUB — Central hub spreadsheet
- SS_ACADEMIC — Student transcripts and academic data
- SS_ADMIN — Administrative data
